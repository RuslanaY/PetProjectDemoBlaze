# DemoBlaze QA Pet Project

## Goal
To create a portfolio-ready QA test project focused on practicing, applying, and improving software testing skills through hands-on work.

## Application Under Test
DemoBlaze e-commerce web app and its public API.

## Scope
- Web UI: core user flows (auth, catalog, cart, checkout), session behavior
- API: catalog, auth, cart endpoints (Postman + Newman)

## Artifacts
- Test Design Overview, requirements, checklist, UI/API test cases: `docs/`
- Postman collection: `postman/collections/`
- Newman reports: `newman/reports/`
- SQL portfolio DB (test management model): `sql/`

## Tools
Postman, Newman, Excel, GitHub, SQL.
